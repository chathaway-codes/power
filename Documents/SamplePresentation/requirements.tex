\subsection{Requirements}

\frame{\frametitle{Functional Requirements}
\begin{itemize}
 \item Allow users to register accounts
 \item Store redundant basic information
 \item Require secure authentication of users
 \item Allow users to apply for loans
 \item First version must allow users to apply for collateralize personal loans and collateralize consumer loans
 \item Allow users to view existing or previous applications
 \item Allow users to cancel applications within the first 24 hours
\end{itemize}
}

\frame{\frametitle{Functional Requirements}
\begin{itemize}
 \item All user interactions will take place in a webpage.
 \item The user will input data through webpage
 \item During a single session, the web page will keep forms populated
 \item To be able to do any interaction, the user must be logged in.
 \item All completed documents will be provided in the form of PDF documents and will be available via downloads.
\end{itemize}
}
